# Build Info Example

Этот пример демонстрирует использование компонента `BuildInfo` для отображения информации о сборке во время выполнения.

## Как использовать

1. Создайте новый GameObject в сцене
2. Добавьте к нему скрипт `BuildInfoExample`
3. Настройте параметры в инспекторе:
   - `Show On Start` - показывать информацию при запуске
   - `Display Key` - клавиша для отображения информации (по умолчанию F1)

## Что делает пример

- Автоматически добавляет компонент `BuildInfo` если его нет
- Отображает информацию о сборке в консоли при запуске (если включено)
- Позволяет показывать информацию по нажатию клавиши
- Показывает краткую информацию в GUI во время игры

## Полезно для

- Отладки и тестирования различных сборок
- Проверки версий в билдах
- Диагностики проблем с платформами
