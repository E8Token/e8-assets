# Energy8 Google Package Manager

Автоматический менеджер для управления Google Unity пакетами с возможностью установки, уведомлений о обновлениях и управления пакетами через Unity Package Manager.

## Возможности

- 🔄 Автоматическое обновление списка доступных Google Unity пакетов
- 📦 Установка и удаление пакетов одним кликом
- 🔔 Уведомления о доступных обновлениях
- ⚙️ Гибкие настройки автообновления
- 📱 Поддержка всех категорий Google пакетов:
  - Firebase (Authentication, Firestore, Analytics, и др.)
  - Android (Performance Tuner, App Bundle)
  - Google Play (Instant, Integrity API, Asset Delivery)
  - AR (ARCore Extensions)
  - Tools (External Dependency Manager)
  - Advertising (в планах)

## Установка

1. Скопируйте папку `io.energy8.google-package-manager` в папку `Packages` вашего Unity проекта
2. Unity автоматически обнаружит и импортирует пакет
3. Пакет появится в меню `Energy8` в верхней панели Unity

## Использование

### Основное меню

Все функции доступны через меню `Energy8 → Google Package Manager`:

- **Open Manager** - Открыть главное окно менеджера пакетов
- **Refresh Package List** - Обновить список доступных пакетов
- **Check for Updates** - Проверить наличие обновлений
- **Settings** - Открыть настройки
- **Clear Download Cache** - Очистить кеш скачанных файлов

### Главное окно менеджера

1. **Категории пакетов** - Выберите категорию в левой части окна
2. **Список пакетов** - Просмотрите доступные пакеты в выбранной категории
3. **Фильтры** - Используйте поиск и фильтры для быстрого поиска
4. **Установка/Обновление** - Нажмите кнопки для управления пакетами

### Настройки

Откройте настройки через меню `Energy8 → Google Package Manager → Settings`:

- **Auto Update Settings** - Настройки автообновления
- **Download Settings** - Параметры скачивания
- **Notification Settings** - Управление уведомлениями
- **Advanced Settings** - Дополнительные параметры

## Технические детали

### Поддерживаемые форматы пакетов

1. **.tgz** - Архивы пакетов, устанавливаются через Package Manager

### Кеширование

- Скачанные .tgz файлы сохраняются в `Library/GooglePackageCache`
- Unity ссылается на локальные .tgz файлы через file: URL
- Кеш можно очистить через меню или настройки

### Автообновления

- Автоматическая проверка обновлений при запуске Unity (опционально)
- Периодические проверки обновлений (настраиваемый интервал)
- Уведомления о доступных обновлениях

## Устранение проблем

### Пакет не устанавливается

1. Проверьте подключение к интернету
2. Убедитесь, что Unity может записывать в папку проекта
3. Проверьте логи Unity Console на наличие ошибок
4. Попробуйте очистить кеш через меню

### Медленная загрузка

1. Увеличьте время ожидания в настройках
2. Проверьте настройки прокси/файрвола
3. Попробуйте скачать пакет позже

### Проблемы с обновлениями

1. Используйте "Refresh Package List" для принудительного обновления
2. Проверьте настройки автообновления
3. Убедитесь, что включены уведомления

## Известные ограничения

- Поддерживаются только .tgz пакеты
- Некоторые пакеты могут требовать дополнительной настройки
- Интернет соединение необходимо для первоначального скачивания
