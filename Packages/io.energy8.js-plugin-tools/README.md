# Energy8 JS Plugin Tools

A modular Unity WebGL plugin system that enables rapid and clean integration of JavaScript plugins into Unity via a common communication framework.

## Modules

- **Core**: Provides the fundamental architecture and APIs for JavaScript plugin integration in Unity WebGL.

## Features

- **Plugin Registration**: Register C# plugins that interface with JavaScript.
- **JS ↔ C# Communication**: Robust two-way messaging system.
- **Modular Design**: Easily extend with new modules.
- **Debug Mode**: Inspect internal state during development.

## Requirements

- Unity 2021.3 LTS or newer
- Target platform: WebGL
